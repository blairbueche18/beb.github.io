My first project is a game meant to mimic the arcade game “Galaga.” This game was meant to be played with two buttons and a joystick on an arcade machine, but I coded it to be used with the arrow keys. The premise of the game is to move the ship back and forth to aim at the enemies. Next, you can shoot a bullet at the enemies using the up arrow. If you are successful, you will hit an enemy ship and kill it. However, if an enemy ship touches you or is able to shoot you, you will lose a life and restart the stage. An extra life can be gained back by getting 100 points. If all lives are lost, the game will end and you will have to restart. I tried to make the game as similar to the actual arcade as possible by making stars in the background and using images of the enemies and spaceship from online.
